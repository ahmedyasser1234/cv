## Overview
A high-end, pixel-perfect professional portfolio and digital CV designed to showcase expertise in Web Application Development, Motion Graphics, and Design. Built with modern web technologies, featuring a cinematic dark mode, seamless RTL/LTR bi-directional support, and interactive glassmorphism components.

## Features
- **Luxury Dark Theme:** A premium, fully custom dark mode aesthetic with localized glow effects.
- **Bi-directional Support (RTL/LTR):** Seamless toggling between Arabic and English utilizing `react-i18next`.
- **Responsive Layout:** Perfectly adapted for mobile, tablet, and desktop viewing.
- **Interactive UI:** Smooth 'Hover Lift' animations, dynamic background grids, and interactive skill modals using `Framer Motion`.
- **Dynamic Projects Showcase:** A visually stunning grid layout integrating real-world project links and customized mockups.

## Technologies Used
- **Frontend Framework:** React (Vite)
- **Styling:** Vanilla CSS & Glassmorphism UI
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Internationalization:** i18next

## Setup Instructions

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ahmedyasser1234/cv.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd cv
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```

## Deployment
This project is optimized for deployment on platforms like Netlify or Vercel. 
To build the production bundle, run:
```bash
npm run build
```
Then deploy the `dist/` folder.
